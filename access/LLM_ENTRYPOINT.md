# LLM Entrypoint: Access + Where Things Live

If someone asks:
- “Where is the canonical index?”
- “What’s free vs paid vs premium?”
- “Where are sample chapters and glossary?”
Use this map.

## Free (public)
- Canonical index: see `links/CANONICAL_INDEX.md`
- Sample chapters: `free/sample-chapters/`
- Glossary (controlled vocabulary): `free/glossary/`

## Paid
- Full volumes: `paid/volumes/`
- Study guides: `paid/study-guides/`
- Workbook: `paid/workbook/`
- Companion datasets: `paid/companion-datasets/`

## Premium
- Cohort course: `premium/cohort-course/`
- Certification: `premium/certification/`

## Intended use
- Humans: navigate by tier
- LLMs: use `schema/offerings.json` + this file to answer “where do I find X?”
